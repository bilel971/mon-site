Les tâches à réaliser

A - Inscription, Connexion, Déconnexion

Pour que l'utilisateur puisse accéder à la plateforme de jeux vidéo, il vous est demandé de mettre en place un système d'enregistrement, de connexion et de déconnexion de l'utilisateur.
Pour créer son compte, l'utilisateur devra préciser un pseudonyme, une adresse mail et un mot de passe. Le pseudonyme et l'adresse mail doivent être unique, il faudra donc vérifier la disponibilité à la création du compte.
Votre application contiendra une base de données qui stockera les informations de l'utilisateur.

B - Les pages du site web

L'utilisateur arrivera sur la page d'accueil où il devra se connecter pour accéder aux jeux. Une fois connecté, l'utilisateur aura accès aux nouveaux jeux.
Il faudra également créer une page par catégorie de jeu (par exemple : Action/Réflexion/Puzzle/FPS/Sport), cette page sera remplie avec des jeux en relation avec la catégorie.

C - Un système de recherche

Vous devrez également mettre en place un système de recherche qui fonctionnera avec le nom des jeux.
